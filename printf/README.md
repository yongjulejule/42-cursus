# ft_printf

Recode printf function in C

Don't need to work exactly the same to printf but need to recode following conversions
- flags: `+, -, #, ' ', 0`
- width: `[0-9], *`
- precision: `[0-9], *`
- length: `l, ll, h, hh`
- specifier: `csdiupxX%`
